# Daan Reddy A Portfolio

A responsive personal portfolio for Daan Reddy A, an AI-integrated backend and full stack developer.

## Features

- Dark portfolio layout inspired by the provided reference template
- Hero section with profile image, resume download, GitHub, LinkedIn, and contact actions
- About, education, experience, skills, projects, currently working on, services, and contact sections
- Local static assets for the profile photo and resume PDF
- Dependency-free Node.js static server

## Run

```bash
npm start
```

Open:

```text
http://localhost:3000
```
